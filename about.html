About Our Spotify Analysis Project

Project Overview

This analysis project explores the relationship between music characteristics and temporal patterns throughout the year. Using data from Spotify's API, we investigate how audio features of songs vary across seasons and months, providing insights into musical trends and listener preferences in 2018.

Data Source & Methodology

Our data comes from the Spotify Web API, which provides detailed audio features for millions of tracks. We focused on 2018 data, analyzing seven key audio features that Spotify calculates for each song. The analysis was conducted using Python, with visualizations created using matplotlib, seaborn, and Altair.

Key Audio Features Analyzed

  • Acousticness: A confidence measure of whether the track is acoustic vs. electronic
  • Danceability: How suitable a track is for dancing based on tempo, rhythm stability, and beat strength
  • Energy: Represents the intensity and activity level of the track
  • Valence: The musical positiveness conveyed by the track
  • Instrumentalness: Predicts whether a track contains no vocals
  • Liveness: Detects the presence of a live audience in the recording
