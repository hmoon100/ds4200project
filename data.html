Spotify Playlist Patterns

How Seasons and Weather Influence Spotify Users' Listening Habits

Introduction Data Visualizations Summary

Introduction to Data

Our data analysis draws upon the Spotify Web Developer API and the OpenWeather API, offering a comprehensive dataset spanning between 2,000 and 5,000 records. The Spotify API provides access to top tracks, playlists, and genre popularity scores, while the OpenWeather API contributes seasonal and weather data. By combining these datasets, we aim to uncover the relationship between music listening habits and environmental factors.

Data Collection

Data collection involves retrieving information on top tracks and playlists across various genres, including play counts and listener demographics. The Spotify API allows us to gather detailed data about genre popularity over time, while the OpenWeather API supplies weather conditions such as temperature during the same timeframe. Preprocessing steps will include normalizing formats, handling missing values, and merging datasets for unbiased and accurate comparisons.

Analysis Plan

Our analysis will focus on identifying temporal trends in genre popularity across different seasons and weather conditions. We will use statistical methods such as correlation coefficients and regression models to examine relationships between weather variables and music preferences. Clustering algorithms will group genres with similar seasonal patterns, and demographic segmentation will provide insights into listener variations. Findings will be presented using static visualizations like line graphs, bar charts, and heatmaps, as well as interactive dashboards using tools like Plotly.
